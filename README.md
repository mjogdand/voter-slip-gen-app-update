# Voter Slip Generator App

Welcome to the official release repository for the **Voter Slip Generator**. This professional Windows application is designed for efficient Electoral Roll processing, allowing users to generate high-quality voter slips from Excel data with seamless PDF export.

---

## 🚀 How to Download & Install

### 1. Download the Installer
Navigate to the [Releases](https://github.com/mjogdand/voter-slip-generator-app/releases) section of this repository and download the latest setup file:
*   **File Name:** `VoterSlipGenerator_Setup_2.1.0.exe`
*   **Platform:** Windows 10/11 (x64)

### 2. Installation Steps
1.  **Run the Installer:** Double-click the downloaded `.exe` file.
2.  **Permissions:** The installer requires Administrator privileges. Click "Yes" if prompted by Windows User Account Control (UAC).
3.  **Follow the Wizard:** Choose your installation directory and click "Install".
4.  **Launch:** Once complete, you can launch the application from the desktop shortcut or the Start Menu.

## ✨ Key Features
*   **Excel to Slips:** Convert voter data directly from Excel sheets into formatted slips.
*   **PDF Processing:** Built-in tools for PDF to Excel conversion and cropping.
*   **Multi-Language Support:** Optimized for various Indian languages using Noto Sans fonts.
*   **Offline Capability:** Robust local fallback system ensures the app works even without an internet connection.
*   **Professional Layouts:** Supports multiple page sizes (A4, A3, Letter, Legal) and orientations.
